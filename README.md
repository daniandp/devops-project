This is Term project for DevOps Term03 - CPRO by Lambton College.

Project Title: Login Page for Appliance Store
Brief Description of Project: Our project will be a login screen, so you can login to our appliance store. It will be connected to a simple CSS and HTML page. Login page will prompt for username and password.

This application uses NodeJS, ExpressJS, MongoDB, Pug templates. This project uses Azure VM for automation deployment via GitHub.

Features:
Register
Login
User Profile
User Dashboard
