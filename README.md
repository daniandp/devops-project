=====================================================================
Project Title: Login Page for Appliance Store
=====================================================================
Brief Description of Project: Our project will be a login screen, so you can login to our appliance store. It will be connected to a simple CSS and HTML page. Login page will prompt for Email and password.

=====================================================================
TechStack
=====================================================================
This application uses NodeJS, ExpressJS, MongoDB, Pug templates. This project uses Azure VM for automation deployment via GitHub.

=====================================================================
Features:
=====================================================================
Register,
Login,
User Profile,
User Dashboard

=====================================================================
Follow these steps to run project
=====================================================================

1. install npm v11.6
2. install nodejs v20
3. npm install (install dependencies)
4. node app.js or npm start (start server)
